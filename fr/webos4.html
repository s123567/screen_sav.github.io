Choisissez le menu Démarrage rapide

Choisissez votre langue
Laissez les 2 types de rotations d'écran sur OFF
IMPORTANT : Réglez l'écran à la bonne heure

Appuyez sur la touche SETTINGS, et allez dans le menu GENERAL, puis RESEAU

Vous ne pourrez pas installer l' application, si votre écran n'est pas relié internet

Ne reliez pas votre écran à un Wifi pour invités (ce genre de réseau n'a pas assez de permissions pour permettre un fonctionnement optimal)

Choisissez le type de connection à paramétrer (câblée ou Wifi)

Connexion Câblée
Si vous n'êtes pas sur un réseau avec IP fixes, vous serez connecté automatiquement en branchant votre câble
Sinon, renseignez les données réseaux indiqués par votre siège ou votre prestataire de service internet.

Wifi : choisir le réseau désiré et rentrez son mot de passe

Appuyez sur la touche SETTING, allez dans EZ SETTINGS, puis SI SERVER SETTING

Choisissez SI SERVER SETTING

Ajustez les paramètres suivants
=> Nom de domaine qualifié = ON
=> Entrez cette URL 'http://webos.deepidoo.com/dj/ipk'
=> Application Launch Mode = LOCAL
=> Application Type = IPK

Cliquez sur DISTANT, puis confirmez

Redémarrez votre écran avec votre télécommande

Appuyez sur HOME button, trouvez l'icone DJ et appuyez sur OK

Un code à 4 chiffres vous sera demandé. Renseignez le code donné par Deepidoo

L'adresse de votre site est indiquée. Confirmez en appuyant sur la touche 0, SEULEMENT si l'adresse est correcte. Dans le cas contraire, appelez Deepidoo. NOTE: Si besoin, vous pouvez redémarrer cette étape en appuyant sur le bouton STOP
