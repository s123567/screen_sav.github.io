Ouvrir le menu caché en laisant appuyé la touche SETTNGS jusqu'à l'apparition d'une bulle en haut à droite de l'écran
Appuyez rapidement 4 fois sur le bouton 0, puis sur OK

Ajustez les paramètres suivants ainsi
AUTO POWER = OFF
SPEAKER = OFF

=> Application Type = IPK
